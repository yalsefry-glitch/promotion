## النشر على Render

### 1) أنشئ Web Service
- render.com ← **New** ← **Web Service**
- اربط GitHub واختر repository باسم `promotion`

### 2) الإعدادات
- **Runtime / Language**: Node
- **Build Command**: `npm install`
- **Start Command**: `npm start`
- **Instance Type**: Free

### 3) المفاتيح (الأهم) — تبويب Environment
أضف متغيّرين:
- `GEMINI_API_KEY` = مفتاح جيميني
- `FAL_KEY` = مفتاح فال

احفظ. Render ينشر تلقائياً.

### 4) افتح من الجوال
- Render يعطيك رابطاً مثل `https://promotion-xxxx.onrender.com`
- افتحه في Chrome ← قائمة المتصفح ← **إضافة إلى الشاشة الرئيسية**

⚠️ الطبقة المجانية تنام بعد 15 دقيقة خمول. أول فتحة بعد النوم تنتظر 30-60 ثانية، بعدها تشتغل عادي. طبيعي.

## التشغيل على ويندوز لاحقاً (اختياري، للاختبار)

```
npm install
set GEMINI_API_KEY=مفتاحك
set FAL_KEY=مفتاحك
npm start
```
ثم افتح `http://localhost:8787`
